# NAME
{{BIN}} \- Cardano blockchain utility

# SYNOPSIS
**{{BIN}}** [options]

# DESCRIPTION
**{{BIN}}** is a command-line utility for interacting with the Cardano
blockchain.
For detailed usage, please run:

  **{{BIN}} \-\-help**

# OPTIONS
See the output of **{{BIN}} \-\-help** for a full list of available options
and arguments.

# EXAMPLES
Run the utility to see available options and arguments:
```sh
{{BIN}} --help
```
